# Food Store - Evaluación 1 Programación III

## Descripción
Aplicación frontend de catálogo y carrito de compras.  
Funcionalidades:
- Catálogo dinámico con búsqueda y filtros por categoría
- Carrito persistente en localStorage
- Vista de carrito con productos y total

## Instalación y ejecución
1. Abrir la carpeta `CARBAJAL_GABRIEL_PARCIAL1` en VS Code.
2. Instalar dependencias:
   Powershell:
   pnpm install
